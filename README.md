# To-Do List Project

Este é um projeto simples de uma To-Do List utilizando **HTML**, **CSS** e **JavaScript**.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas da lista
- Interface simples e responsiva
